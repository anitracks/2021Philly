1 """!
2  @file cleanPlaintiffs.py
3  @brief Cleans the plaintiff names and counts frequency of occurance
4  @details The functions in this script can clean plaintiff names and
5  then cluster the names to put similar names together. For instance,
6  - Victoria Fire and Casualty Co. a/s/o Robert Logan
7  - Victoria Fire and Casualty Company a/s/o Francisco Mulero
8 both refer to the came plaintiff and should be clustered together.
9 
10 We did the following to clean the plaintiff names:
11  - Removed "a/s/o", "as subrogee of", “aso”, commas, and the text that followed those items
12  - Removed any non-alphanumeric characters
13  - Used CleanCo to remove LLC, INC, etc.
14  - Put the names into all uppercase
15 
16 Clustering plaintiffs is started by using the [jellyfish](https://github.com/jamesturk/jellyfish)
17 jaro_similarity function as can be seen in the createSimilarityMatrix() function to
18 calculate similarities between all the cleaned plaintiff names. Once the
19 similarity matrix has been created, the [DBSCAN](https://towardsdatascience.com/dbscan-algorithm-complete-guide-and-application-with-python-scikit-learn-d690cbae4c5d)
20 algorithm is used to cluster similar names.
21 
22 Typical use cases are as follows:
23 
24  python .\src\cleanPlaintiffs.py ..\data\MC_specialreport_limiteddaterange_Nov6.csv -s
25  python .\src\cleanPlaintiffs.py ..\data\MC_specialreport_limiteddaterange_Nov6.csv -p .\src\20215121490-comparisonMatrix.pkl
26 
27 The first creates the similarity matrix (may take 9 minutes). The second uses this
28 similarity matrix to build a plaintiff vs outcome matrix.
29 
30  @author Seth McNeill
31  @date 2021 March 11
32 """

244 def dbscanFit(fname, eps, min_samples=2):
245  """! Runs DBSCAN clustering on the similarity matrix in fname
246  with epsilon (eps) set to eps and clusters requiring at least
247  min_samples in them.
248 
249  @param fname The filename for a pickle file containing a similarity matrix
250  @param eps Epsilon - the largest distance allowed to be considered a neighbor
251  @param min_samples The minimum number of names in a cluster
252  """
253  similarNames = pd.read_pickle(fname)
254  distanceNames = 1 - similarNames
255  dbfit = DBSCAN(eps=eps, min_samples=min_samples, metric='precomputed').fit(distanceNames)
256  return dbfit
