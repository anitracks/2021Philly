\doxysubsection{Detailed Description}
Cleans the outcome data and counts how many of each there are. 

The main function reads in the name of a csv file to process. The \char`\"{}\+Case Outcome\char`\"{} column is processed via 
\begin{DoxyImageNoCaption}
  \mbox{\includegraphics[width=\textwidth,height=\textheight/2,keepaspectratio=true]{dot_inline_dotgraph_1}}
\end{DoxyImageNoCaption}
 It is cleaned via the following processes\+:
\begin{DoxyEnumerate}
\item Check for nan which arise from blank or Null input values
\item Remove all trailing ALL CAPS WORDS
\item Remove everything after the first period (.)
\item Remove date/time and everything after it (typical format 07/17/2017 1\+:15)
\end{DoxyEnumerate}

Then the count of each unique value is calculated and printed.
