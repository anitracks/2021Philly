Detailed Description

Cleans the plaintiff names and counts frequency of occurance.

The main function reads in the name of a csv file to process.

Todo:

It is cleaned via the following processes:

  1. Check for nan which arise from blank or Null input values
  2. Remove all trailing ALL CAPS WORDS
  3. Remove everything after the first period (.)
  4. Remove date/time and everything after it (typical format 07/17/2017 1:15)

Then the count of each unique value is calculated and printed.
